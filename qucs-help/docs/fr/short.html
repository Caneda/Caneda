Brève description des man½uvres

Général

(valide dans tous les modes)
roulette de la souris
 Ascenseur. Vous pouvez sortir de la page
roulette + Shift
 Tapis roulant. Vous pouvez sortir de la page
roulette + Ctrl
 Modification de l'échelle du dessin


Mode «séléction»

(Menu : Modifier -> Sélectionner)
bouton gauche
 Sélectionne l'élément pointé ; s'il y en a plusieurs, cliquez jusqu'à sélectionner le bon
En gardant le bouton pressé, vous pouvez déplacer le ou les élément(s) sélectionné(s)
Cliquer sur une zone vide et laisser appuyé dessine un rectangle de sélection
Un graphique peut être recadré en cliquant sur l'un de ses coins, en maintenant appuyé et en déplaçant la souris
bouton gauche + Ctrl
 Ajoute / retranche un élément à une sélection déjà existante ; également valable en mode rectangle
bouton droit
 Cliquer sur un fil sélectionne un segment plutôt que l'équipotentielle entière
Double-clic droit
 Ouvre une boîte pour modifier les caractéristiques d'un objet (nom, valeur des paramètres...)


Mode d'insertion

(Choisir un composant dans la partie gauche de l'écran)
clic gauche
 Place une nouvelle instance du composant sur le dessin
clic droit
 Fait pivoter le composant


Mode « fil »

(Menu: Insérer -> Équipotentielle)
clic gauche
 Indique le début (la fin) d'un fil
clic droit
 Inverse la direction d'un angle du fil
double-clic droit
 Termine un fil en l'air


Mode « Coller »

(Menu: Modifier -> Coller)
clic gauche
 Colle le contenu du presse-papier sur le schéma
clic droit
 Pivote les éléments


Souris dans l'onglet « Contenu »


clic gauche
 Sélectionne le fichier
double-clic Ouvre le fichier
clic droit
 Affiche un menu contextuel (pop-up)


Clavier

Qucs comprend de nombreux raccourcis clavier, dont la majeure partie est affichée dans les entrées de menu correspondantes. En voici cependant quelques une de plus :
"Delete" or "Backspace" Supprime le composant sélectionné ou entre dans le mode suppression si aucune sélection n'est active
Flèche gauche ou droite
 Change la position des inspecteurs sélectionnés sur la courbe
Si aucun inspecteur n'est sélectionné, translate la courbe sélectionnée
Si rien n'est sélectionné, sert de tapis roulant
Flèche haute / basse
 Change la position des inspecteurs dans des diagrammes multi-variables
Si aucun inspecteur n'est sélectionné, translate la courbe sélectionnée
Si rien n'est sélectionné, sert d'ascenseur
Tabulation Avance au document ouvert suivant (onglet de droite)
