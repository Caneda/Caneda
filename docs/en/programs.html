Installed Files

The Caneda system needs several files. These are installed during the installation process. The path of Caneda is determined during the installation (configure script). The following explanations assume the default path (/usr/local/).

/usr/local/bin/caneda     - the GUI
/usr/local/share/caneda/bitmaps - contains all bitmaps (icons etc.)
/usr/local/share/caneda/docs    - contains HTML documents for the help system
/usr/local/share/caneda/lang    - contains the translation files

Command line arguments

caneda [file1 [file2 ...]]
