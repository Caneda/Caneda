Caneda
Open source EDA software

Índice de la Ayuda

Caneda es una suite de software EDA open source enfocado en la facilidad de uso y portabilidad. Su objetivo principal es permitir el proceso de diseño completo: captura de esquemáticos, simulación y layout de circuitos o PCBs.

Caneda apunta a soportar múltiples tipos posibles de simulaciones de circuitos, como por ejemplo DC, AC, parámetros S y análisis de distorsión armónica.
